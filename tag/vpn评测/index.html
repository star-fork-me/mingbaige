VyprVPN的服务商是位于美国旧金山的Golden Frog公司，该公司创立于1994年，除了开发VyprVPN，他们还提供一种名字叫CYPHR的免费加密聊天服务，公司的目标是保持一个开放和安全的互联网，反对互联网封锁，同时尊重用户隐私。

最近几年国内开始互联网封锁以来，VyprVPN以其快速稳定的质量深受中国用户的喜爱。那么，VyprVPN在如今全面禁止VPN的大背景下在国内好不好用呢？为此我们对VyprVPN进行了一次全面的测试。

一、客户端支持

在客户端的安装上非常简单，软件下载之后一键安装，整个过程不超过3分钟，VyprVPN支持Windows，Mac OS X，Android和iOS等主流平台上使用，都提供独立的客户端：

VyprVPN支持的协议包括PPTP、L2TP/IPsec、OpenVPN和Chameleon，除PPTP，其他都使用256位加密。所谓Chameleon,中文即变色龙，是Golden Frog开发的一个独立协议，可以用未经修改的OpenVPN256位协议，把VPN流量进行伪装，不会被防火长城认为是VPN连接而被阻止，从而更好的帮助用户安全翻墙。

在windows系统安装VyprVPN软件时，如果你的电脑有些程序运行需要的组件没装过（比如windows .NET Framework 4.5和VC++ 2010），会自动帮你下载安装，由于连接的是国外的下载服务器，下载安装过程会有点慢，请耐心等待安装完成。或者自行百度下载安装组件之后，再安装Vypr的程序。

在安装过程中，如果出现360安全卫士等安全软件提示“修改系统设置”（因为适配器的安装本来就要修改系统设置），要全部允许，不然组件和Vypr程序不能安装。

如果安装卡住了不能继续，自行百度下载安装组件，一般是windows .NET Framework 4.5和VC++ 2010，这2个组件安装之后再安装Vypr程序。

另外，如果大家在购买和安装过程中有什么困难或是不懂的问题，欢迎在本文下方的留言区给我们留言。
